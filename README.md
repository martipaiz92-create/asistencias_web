# Resumen del proyecto

Mi página es un sistema web para tomar asistencias en el colegio.
Tiene dos tipos de usuarios: preceptor y admin.
El preceptor puede marcar a los alumnos como presentes, ausentes o tarde y guardar la asistencia del día.
El admin puede ver los reportes por fecha de todas las asistencias guardadas.
El sistema usa HTML, JavaScript y TailwindCSS para el diseño, y guarda los datos con localStorage, sin necesidad de base de datos.
